# US-OS Vayan Global

A unified global sector operating system dashboard mapping table with 16 ASOs, multi-theme UI/UX, and Next.js 16 App Router.

## Features

- 16 Autonomous Sector Operating Systems (ASOs) with public pages, user pages, dashboards, and assigned themes.
- Multi-theme UI/UX system.
- Role-based access control for super-admin, sovereign-owner, admin, investor, employee, franchise, promoter, developer, and freelancer.
- Authentication with NextAuth.js.
- Database with Prisma and PostgreSQL.
- Caching with Redis.
- Notifications with Novu.
- AI integrations for valuations and predictions.
- Chatbot and ad management.

## Prerequisites

- Node.js 18+
- PostgreSQL database
- Redis server
- NPM or Yarn
